Privacy Policy

Last updated: April 18, 2026
FlipRadar is designed with privacy in mind. We do not collect, store, or sell your personal data.

What FlipRadar does

FlipRadar is a browser extension that reads car listing information from the page you are currently viewing and sends it to the Anthropic Claude AI API to generate a deal analysis. The analysis is displayed directly in your browser.

Data we collect

FlipRadar does not collect any personal data. Specifically:

  • We do not collect your name, email address, or any identifying information
  • We do not store your browsing history
  • We do not track which listings you view or score
  • We do not share any data with third parties other than what is described below

Your Anthropic API key

FlipRadar requires an Anthropic API key to function. This key is stored locally in your browser using Chrome's built-in secure storage (chrome.storage.sync). It is never transmitted to FlipRadar's servers — it is sent directly from your browser to Anthropic's API when you score a listing.

Your API key usage is governed by Anthropic's Privacy Policy.

Listing data

When you click "Score This Deal," FlipRadar reads the visible text content of the car listing page you are viewing and sends it directly to the Anthropic API for analysis. This data is:

  • Sent directly from your browser to Anthropic — not through any FlipRadar server
  • Not stored by FlipRadar in any database
  • Subject to Anthropic's data handling policies

Permissions

FlipRadar requests the following browser permissions:

  • activeTab — to read the listing content on the current page
  • storage — to save your API key locally in your browser
  • host permissions for supported sites (Facebook, Craigslist, OfferUp, AutoTrader, Cars.com) — to inject the scoring button on listing pages

FlipRadar does not request access to your browsing history, bookmarks, downloads, or any other browser data.

Third-party services

FlipRadar integrates with the following third-party services when you use specific features:

  • Anthropic API — processes listing text to generate deal analysis
  • Kelley Blue Book — opens in a new tab when you click "KBB Value" (no data sent automatically)
  • Carfax — opens in a new tab when you click "Carfax" (no data sent automatically)

Children's privacy

FlipRadar is not directed at children under 13 and does not knowingly collect information from children.

Changes to this policy

We may update this privacy policy from time to time. Any changes will be reflected on this page with an updated date.
